Website Details

This section is about how I made this site and other small tidbits about it.

How this whole thing started was due to me wanting to work on a smaller project while I was working on cars and other bits. So I figured making a website to catalogue all of the stuff I done be it hobbies or programming might help. The whole website is built with css and html theough githubpages and I got a template from petoKrisa on github but tweaked numerous things to fit something you'd see in a PlayStation portable/PS3 menu. I prefer this frutiger/skeumorphism aesthetic over the clean metro look partly due to nostalgia but also because I dislike the "metro" overtly clean look a lot of apps have gone through.
